1.Работа с историей изменений:
	1.git log --pretty=format:"%an :%s" --since=3.hours develop-feature1 ^develop ^master --date-order 

	//pretty=format отвечает за форму вывода каждого коммита(Автор: Сообщение), since определяет за какой промежуток времени выводить коммиты, "develop-feature1 ^develop ^master" говорит что cледует выводить коммиты доступные из ветки develop-feature1 но не из master и develop, date-order упорядочивает коммиты по дате%//

	2.git log --grep-231 --pretty=format:"%an: %s (%ad)" --date-order 

	// grep находит все коммиты в которых есть совпадение по строке "231", pretty=format выводит лог в виде "Автор: Сообщение (дата)", --date-order выводит коммиты в хронологическом порядке начиная с самого нового.

2.Избирательное слияние:

	git co develop
	git merge develop-feature1^^^  

	 // Сначала переключаемся на ветку develop затем проводим слияние с четвертым коммитом с конца(develop-feature1 указывает на последний коммит ветки develop-feature1, develop-feature1^ на предпоследний, ^^ на пред-предпоследний и так далее до нужного нам коммита.)

3.Исправление ошибок(Метод взял тут https://git-scm.com/book/ru/v1/Инструменты-Git-Перезапись-истории#Изменение-сообщений-нескольких-коммитов )
	
	git rebase -i HEAD~2 // Запуск этой команды выдаст в текстовом редакторе список коммитов, для которых будет выполнятся определенный сценарий
// Содержимое текстового редактора
	pick f7f3f6d Предпоследний коммит
	pick 310154e Последний коммит
//
	чтобы остановиться на нужных коммитах необходимо изменить pick на edit

// Содержимое текстового редактора после редактирования
	edit f7f3f6d Предпоследний коммит
	edit 310154e Последний коммит
//
 
	после сохранения и выхода для каждого коммита пишем:

	git commit --amend // В текстовом редакторе изменяем текст сообщения коммита
	git rebase --continue //Для перехода к следующему коммиту или для завершения релдактирования после последнего коммита.

	git push -f --set-upstream origin develop-feature3 // пушим все изменения на сервер с параметром -f
